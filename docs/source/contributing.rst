Contributing to DyNet
====================

DyNet is an open source project that is only possible because of contributions from users like you!
We greatly welcome any problems, whether they are to correct a bug or add a feature, and these should be made through a pull request on the `DyNet github page <http://github.com/clab/dynet>`_.
